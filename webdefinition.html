Web Definition

Website

A website is a collection of web pages which includes a set of information and data , accessible publicly and organised by one domain name.

WWW

( World Wide Web ) WWW commonly known as web which consists of all the public websites connected to the internet world wide.

HTTP

( hypertext transfer protocol ) it's a fundamental protocol that allows communication between web users. It is used for transferring data from server to browser.

HTTPs

the “s” in https stands for secure in which your communication and data is secured or encrypted. It provides safety for your websites.

Client Server

client -server is a relationship between client & server. When a client needs to access desired files, it requests the data from a server.

Domain

Domain is one type of internet address or location of a website. Ex. .com, .org, .gov.in

Host

A computer or server connected to a network and providing facilities to other computers and their users or clients.

Hosting

Hosting is an online service that provides you to publish your websites , blog or content on the internet.

FTP

(File Transfer Protocol) FTP is a way to download, upload and transfer computer files from one location to another on a computer network.

Path

A path defines the location of a file or folder in a computer’s system .They often include one or more directories that describe the path to the file or folder.

Extension

The extension is a three or four letter abbreviation that defines the file type.

Editor/vscode

Visual studio code is a software that provides basic support for HTML programming. It also includes snippets, intelligent code editing.

URL-URI/URN

( Uniform Resource Locator ) ( Uniform Resource Identifier ) ( Uniform Resource Name ) A URI is a standard method for identifying resources on the internet by their location name or both. URN and URL together known as URI.

Address bar

Address bar is one type of tool bar which helps you to get access to any website through your default browser.

Title bar

It displays the title of the currently open document or application.
